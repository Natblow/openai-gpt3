# openai-ruby-cli

## Usage

create an `.env` file on the root directory and put your OpenAi Token :

```
OPENAI_ACCESS_TOKEN = 'secret-token'
```

Run the app on your CLI

```
./gpt.rb
```

Then you can get started. You can prompt anything.
